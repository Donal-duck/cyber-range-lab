# Diagrams

Visual documentation of the lab architecture.

## Recommended Tools
- **draw.io / diagrams.net** — free, offline-capable, exports to PNG/SVG
- **Excalidraw** — for hand-drawn-style illustrations

## Conventions
- Commit the source file (`.drawio` XML) alongside the exported PNG
- Keep zone colours consistent: LAN=blue, OPT1=grey, OPT2=red, WAN=green
